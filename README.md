# Matisse

一个用 Jetpack Compose 实现的 Android 图片视频选择框架

- 解决了多个系统兼容性问题
- 按需索取权限，遵循最佳用户实践
- 完全用 Kotlin & Jetpack Compose 实现
- 支持多种拍照策略，可以自由定义拍照逻辑
- 支持自定义图片加载框架，可以自由定义加载逻辑
- 支持同时选择图片和视频，或者单独选择两者之一
- 支持精细自定义主题，提供了日夜间两套默认主题
